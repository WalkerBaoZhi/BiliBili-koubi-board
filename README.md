## ✨ 功能

- 🪙**开箱即用**：当你看到一个视频觉得 `半夜刷到这个以为自己活到头了` 就可以点击视频右下角的扣币按钮来“扣掉”UP主的币
- 📊**实时榜单**：点击插件图标可以查看 **日榜、周榜、月榜、总榜**的扣币排行

## 🚀 安装教程

~~由于本人能力限制~~现在只开发了**Chrome / Edge扩展**

转到 [Releases页面](https://github.com/WalkerBaoZhi/BiliBili-koubi-board/releases) 下载最新版本的zip压缩包

**打开浏览器扩展页面**

1. 打开开发者模式
2. 将下载的zip包拖到浏览器中完成安装
3. 启用扩展
