# 🎯 B站扣币榜（全网通缉版）

这是一个由浏览器插件自动上报的 **全网扣币榜**。

当用户觉得某个视频离谱时，可以点击插件的“扣币”按钮，扣掉 1～2 个“精神币”，并将记录上传到本仓库。

所有扣币数据都公开透明，任何人都可以查看、分析、制作榜单。

---

## 📊 实时排行榜（GitHub Pages）

👉 https://你的用户名.github.io/bilibili-koubi-rank/

---

## 📦 数据文件

所有扣币记录都存储在：

- [`data.json`](./data.json)

结构如下：

```json
{
  "records": {
    "av号": {
      "title": "视频标题",
      "cover": "封面URL",
      "total": 12,
      "history": [
        { "time": 1710000000000, "amount": 2, "reason": "离谱" }
      ]
    }
  }
}
```
